## 1. Resources

## 2. How to setup?

### 2.1 Install library

```bash
yarn init -y
yarn add -D typescript ts-node jest ts-jest @jest/types @types/jest

npx ts-jest config:init
```

### 2.2 Test structure

- arrange, actual, assert
